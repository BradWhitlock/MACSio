This is a weak scaling study of MACSio that shows how different I/O interfaces perform when writing data out for different mesh types. MACSio was run on LLNL's Surface cluster to demonstrate the performance of the Conduit/Relay output plugin for MACsio. The performance was measured for 4 Conduit protocols and compared to MACSio's HDF5 serial interface. MIFMAX trials run MACSio so it will generate a separate file for each MPI rank at each time step. SIF trials generate a single file for each time step. Note that in SIF mode, data are lacking for certain protocols that are only able to add data to the file by reading back and appending the full data in serial.

  • MIFMAX 2D uniform
  • MIFMAX 2D rectilinear
  • MIFMAX 2D curvilinear
  • MIFMAX 2D unstructured
  • MIFMAX 2D arbitrary
  • MIFMAX 3D uniform
  • MIFMAX 3D rectilinear
  • MIFMAX 3D curvilinear
  • MIFMAX 3D unstructured
  • MIFMAX 3D arbitrary
  • SIF 2D uniform
